# Superstore Sales Analytics (SQL + Power BI)
End-to-end analytics project: SQL data modeling + Power BI dashboard for Superstore e-commerce dataset.

## Tech Stack
SQL Server: Normalized tables (Customers, Products, Orders, Calendar) + fact view vw_FactSales

Power BI: Star schema model, KPI cards, time series, slicers by Year/Region/Segment

## Key Features
NetSales = Sales * (1 - Discount/100) (discount as %)

Profit Margin = Profit / Sales
